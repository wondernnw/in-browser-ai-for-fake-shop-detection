## Structure

The monorepo is devided in three parts, each in a folder:

- `random_forest`: Contains code for building the RF model ([README.md](./random_forest/README.md)).
- `deep_feedforward_network`: Contains code for building the DFFN model ([README.md](./deep_feedforward_network/README.md)).

Please refer to the README.md file in each folder on setup and execution instructions.
